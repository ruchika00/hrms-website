# HRMS Landing Page (WordPress Custom Theme)

This project is a simple HRMS (Human Resource Management System) landing page built using a custom WordPress theme. It is designed to replicate a modern SaaS-style website layout.

## Features
- Custom WordPress theme
- Responsive design (mobile-friendly)
- Hero section with CTA (Schedule a Demo)
- Feature section (Employee, Attendance, Payroll, Recruitment)
- Clean and professional UI

## Tech Used
- WordPress
- PHP
- HTML & CSS

## Setup
1. Copy project to XAMPP `htdocs`
2. Start Apache & MySQL
3. Open `http://localhost/hrms-site`
4. Activate the custom theme in WordPress
